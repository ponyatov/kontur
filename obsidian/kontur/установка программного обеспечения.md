# установка программного обеспечения

![[el/программное обеспечение]]

## [[Linux]]

```sh
$ sudo apt update
$ sudo apt install -yu git make curl
$ git clone -o lab https://gitlab.com/ponyatov/stm32book.git ~/stm32book
$ cd ~/stm32book
$ make install
```

### update

```sh
$ cd ~/stm32book
$ make updev
```
